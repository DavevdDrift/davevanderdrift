Designing a website for Midusa

Designing a website for fun

Date: 02/08/2021

About the Website Midusa
I wanted to improve and learn more about design, which is why I started making a landing page for fun. I love it to play videogames and I wanted to make a landing page for a game that I’ve played. In this game, people roleplay together in a similair world as Game of Thrones. You got magic, castles, dragons, Kings and a lot of fascinating stories with roleplay. People make these stories into short movies, which they post on YouTube.

The Process
I started making the design with Figma. During my study I learned a lot about using Figma and I also learned a lot about UI and UX. The website should feel magical and medieval, since I make a website about a Kingdom full of wizards. I used the color pink, because it is the main color of that Kingdom. I also used a classical font, to give the website more of a medieval feeling. The background color is black, because it is a pretty dark-themed and scary Kingdom. I used blobs, because they look like particles of a potion. I also used some pixelated particles around those blobs. These particles are the same as in the actual game called Minecraft. The images that I used are related to the content that people make about the game. For example, the map on top has been made by me in the past, which shows the map of the world, including Midusa. I also used thumbnails from the YouTube content about the Kingdom Midusa.

Result
The result is a landing page for a website about The Kingdom Midusa. You can find information about the stories and the world of the Kingdom Midusa. You can also look up the most recent video’s that the content creators of Midusa have made. At last, you can get in contact with the amazing community at Discord and Twitter.
