  • The Wingman Dating app

    Date: 24/04/2019
    Grade: 8.2

  • Issue
    There are few differences between available dating applications when matching users, as users are often judged on their photos. There is insufficient insight into a new way of dating, which focuses on aspects other than visual material. There is also insufficient insight into the functional possibilities of sensors in a smartphone, this also plays a role in the research.

    Objective
    Providing insight into new ways to match users with the same interests. This takes into account the functional possibilities of sensors in a smartphone, with regard to the experience of users of a dating application. These sensors must contribute to the match process.

    Research and Design
    I delivered several works with my project group, which are divided into a research report and a design report.

    The research report contains the competition analysis, Benchmark, research target group, in-depth interviews, personas, empathy map, User needs and concept scenarios.

    The design report contains the user flow, sitemap, customer journey, style guide, patterns & design principles, wireframes, system requirements, usability test and the hi-fi prototype.

    Result & Feedback
    We developed a dating application that can be used to find people with the same hobbies. A buddy system is also used for this. Which is a built-in AI, which helps the user in making choices and taking steps, in order to create the best possible match and result.
