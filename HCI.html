A virtual reality game

Helping people overcome their fear with virtual reality

Date: 10/07/2019 - Grade: 7.5

About this project
In the course "Human Computer Interaction" I got to know a new side of my study. I got in touch with new kinds of special elektronics, programs and equipment. A lot of cool and unique programms and machines can be made with these things. I have worked with many topics during the Human Computer Interaction course. These topics are Wearables, Playful Interactions, Artificial Creatures and Augmented / Virtual Reality.

The virtual Reality Game
Together with my project group I worked on creating a virtual game. We did this by using Unreal Engine 4. The object of this game is to overcome phobias by playing the game. Virtual Reality provides a realistic immersion, which helps create a realistic but same simulation. This caused some fear for many people, but also a lot of fun.

The idea was to make people do something they would never dare to do in real life. That’s how this game tries to help people overcome their fears, in order to improve their lives.
